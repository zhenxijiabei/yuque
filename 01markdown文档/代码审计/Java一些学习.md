开发一个JavaWeb，经历阶段：

1、静态页面

2、CGI通用网关接口

3、Servlet+Tomcat 	

4、Struts

5、spring（maven（整合所有jar包） mvn -v）

6、springmvc（model、view、controller）、

7、spingboot(整合所有框架，包含shiro框架（登录认证）)



SpingBoot基于Spring开发，并非新的技术，默认配置了很多框架。

优点：

- 为所有Sping开发者更快入门
- **开箱即用**，提供各种默认配置来简化项目配置
- 内嵌式容器简化web项目
- 没有冗余代码生成和XML配置的要求
