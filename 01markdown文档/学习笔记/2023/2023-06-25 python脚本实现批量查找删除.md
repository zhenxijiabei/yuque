# 问题

在最初写markdown文档时候，都是用的绝对路径，当时没觉得有问什么问题，后面发现存放图片的文件夹越来越大，图片越来越多，而且混乱不堪，可移植性非常差，索性后面决定都改为相对路径存放图片。

到那时遇到了一个问题，就是之前保存那么多的图片，怎么知道哪些图片被哪些具体文档引用了。想把对应图片都放到具体文件所在目录下的相对路径中，于是就有了下面的脚本。。。



# 解决方案

1、首先把需要清空内容的所有文件名，放到一个dir.txt的文本文件里，可以使用dir /b > dir.txt命令

确保dir.txt和所需清空内容的文本在同一个文件夹下，然后执行 **清空文本内容.py** 脚本

清空文本内容.py

```python
#coding:utf-8
```

此时便已全部清空文档内容。



2、批量删除md文件对应图片，注意我这里图片的绝对路径都是在"d:/Typora文档/typora-pic/"目录下，具体还得根据实际情况来。

批量删除md文件对应图片.py

```python
#coding:utf-8
import re
import os

# 1、定义一个提取文件名的函数1
def extractFileName1(file):  #定义一个提取文件名函数1
    with open(file,'r',encoding='utf-8') as file:   #with open方式打开输入文件
        for line in file:   #每次读取一行
            # print(line)
            rs = re.findall(r"\[image-(.+?)\]",str(line))   #正则匹配出图片名的数字部分
            for i in rs:
                if i is not None:   #判断当前内容不为空时
                    result = "image-"+str(i)+".png"     #拼接图片名
                    print(result)
                    with open('d:\\正则匹配结果.txt','a',encoding='utf-8') as file1: #重新打开一个新文本文件用来保存提取结果
                        file1.write(result+"\n")    #保存提取结果

# 2、定义一个提取文件名的函数2
def extractFileName2(file):  #定义一个提取文件名函数2
    with open(file,'r',encoding='utf-8') as file:   #with open方式打开输入文件
        for line in file:   #每次读取一行
            # print(line)
            rs = re.findall(r"640-(.*?).webp",str(line))   #正则匹配出图片名的数字部分
            for i in rs:
                if i is not None:   #判断当前内容不为空时
                    result = "640-"+i+".webp"     #拼接图片名
                    print(result)
                    with open('d:\\正则匹配结果.txt','a',encoding='utf-8') as file1: #重新打开一个新文本文件用来保存提取结果
                        file1.write(result+"\n")    #保存提取结果

with open("dir.txt",'r',encoding='utf-8') as files:   #with open方式打开输入文件
#需要清空的文本放在dir.txt里面
    for line in files:   #每次读取一行
        # print(line.strip())
        file = line.strip()
        extractFileName1(file)
        extractFileName2(file)

#3、定义一个删除图片的函数
def DelFile():
    with open('d:\\正则匹配结果.txt','r',encoding='utf-8') as fo:     #打开匹配结果
        for m in fo:
            if m is not None:   #判断读取内容不为空
                m = m.strip("\n")   #去除末尾的换行符
                if os.path.exists("d:/Typora文档/typora-pic/"+ m):
                    os.remove("d:/Typora文档/typora-pic/"+ m)
   
DelFile()
os.remove("d:/正则匹配结果.txt")

```
